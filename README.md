# UCS761 - Deep Learning

This repository contains implementations of fundamental deep learning models developed as part of the **UCS761 Deep Learning Laboratory course**.


## Lab Assignments

### Assignment 1 – Perceptron Logic Gates

Design and implementation of a perceptron model to simulate basic logic gates:

- AND
- OR
- NAND
- NOR

File:  
`Assignment1_Perceptron_Logic_gates.ipynb`

---

### Assignment 2 – Single Layer Perceptron Implementation

Implementation of a **Single Layer Perceptron (SLP)** for classification tasks.

File:  
`Assignment2_Implement_single_layer_perceptron.ipynb`

---

### Assignment 3 – Multiple Linear Regression using Perceptron

Implementation of **Multiple Linear Regression** using a perceptron model trained with gradient descent.

File:  
`Assignment3_multiple_linear_regression_using_perceptron.ipynb`

---

### Assignment 4 – Backpropagation Implementation

Implementation of a **Multilayer Perceptron (MLP)** trained using the **Backpropagation algorithm**.

File:  
`Assignment4_Implement_backpropagation.ipynb`
